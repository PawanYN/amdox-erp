# Amdox ERP Frontend

This is the Next.js 15 frontend application workspace for the Amdox ERP Suite.

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Run Development Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
